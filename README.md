# looky-js

A basic webpage boilerplate using [three.js](https://threejs.org/).

## Features
- Minimal setup for a three.js scene
- Simple spinning cube example
- Ready to extend for your own 3D projects

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the development server (Vite):**
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
4. Preview the production build locally:
   ```bash
   npm run preview
   ```

## Project Structure
- `index.html` — Main HTML entry point for Vite
- `src/main.js` — JavaScript entry, initializes three.js scene
